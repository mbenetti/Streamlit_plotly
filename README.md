# Streamlit and plotly visualizations
Useful snipped of code to enbedded Plotly graphics in your streamlit webapp

Clone the repo:

```
git clone https://github.com/mbenetti/Streamlit_plotly.git
```

```
cd Streamlit_plotly
streamlit run app.py

```
